# 🎧 Echo Protocol | Decentralized Audio

[![Network: Ethereum](https://img.shields.io/badge/Network-Ethereum-1DB954.svg)](#)
[![Storage: IPFS](https://img.shields.io/badge/Storage-IPFS-blue.svg)](#)

Echo is a decentralized music streaming platform where artists own their content and listeners tip them directly. By removing the middleman, Echo ensures that 100% of tips go straight to the creator's wallet.

## 🌟 Key Features

- **📡 IPFS Hosting:** Audio files are stored on a decentralized network, ensuring they can never be censored or deleted.
- **💎 Direct Tipping:** Listeners can tip artists in ETH with a single click.
- **🎵 Spotify-Style UI:** A premium dark-mode interface built for music discovery.
- **🔗 On-Chain Metadata:** Track titles and artist addresses are stored immutably on the Ethereum blockchain.

## 🛠️ Setup Instructions

1. **Deploy Contract:** Use `EchoStream.sol` on any EVM chain.
2. **IPFS Upload:** Upload your `.mp3` files to IPFS and get the CID (hash).
3. **Register Track:** Call the `uploadTrack` function with your CID and Track Title.
4. **Launch UI:** Open `index.html` to start streaming and tipping.
